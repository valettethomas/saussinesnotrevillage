Politique de Confidentialité

Votre vie privée est importante pour nous

1. Introduction

Saussines Notre Village s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons, et protégeons vos données personnelles lorsque vous visitez notre site web.

Responsable de traitement : Saussines Notre Village

Email : saussinesnotrevillage@gmail.com

2. Données Collectées

2.1 Données de contact pour les demandes de procuration

Lorsque vous cliquez sur le bouton "J'ai besoin d'une procuration" et nous envoyez un email, nous collectons :

  • Votre adresse email
  • Votre prénom et nom (selon les informations que vous partagez)
  • Votre numéro de téléphone (optionnel)
  • Les informations électorales que vous nous transmettez volontairement

Ces données sont collectées uniquement si vous nous les communiquez explicitement via email.

2.2 Données de navigation

Notre site web peut collecter automatiquement :

  • Votre adresse IP
  • Le type de navigateur et le système d'exploitation
  • Les pages visitées et le temps passé sur chaque page
  • Votre localisation approximative (basée sur l'IP)

Ces données sont collectées via les serveurs d'hébergement et les outils d'analyse (si applicable).

2.3 Données Facebook

Notre site intègre des plugins Facebook (embeds de pages et de flux). Facebook collecte certaines données lorsque vous interagissez avec ces éléments. Veuillez consulter la Politique de confidentialité de Facebook pour plus d'informations.

3. Utilisation des Données

Nous utilisons vos données personnelles pour :

  • Traiter vos demandes de procuration électorale
  • Vous répondre par email
  • Améliorer notre site web et son contenu
  • Analyser l'usage de notre site pour optimiser l'expérience utilisateur
  • Vous envoyer des communications pertinentes concernant les élections (si vous avez donné votre consentement)

4. Base Juridique du Traitement

Le traitement de vos données personnelles repose sur :

  • Votre consentement explicite : Lorsque vous nous envoyez une demande de procuration
  • L'obligation légale : Obligations relatives aux élections
  • Notre intérêt légitime : Améliorer notre site web et communiquer efficacement

5. Partage des Données

Nous ne partageons vos données personnelles avec des tiers que dans les cas suivants :

  • Si la loi l'exige (autorités électorales, autorités publiques)
  • Avec nos prestataires techniques (hébergeurs, services d'email) qui traitent vos données en notre nom
  • Avec votre consentement explicite

Vos données ne sont jamais vendues à des tiers à des fins commerciales.

6. Sécurité des Données

Nous mettons en place des mesures de sécurité techniques et organisationnelles pour protéger vos données contre :

  • L'accès non autorisé
  • La modification non autorisée
  • La divulgation non autorisée

Vos demandes de procuration sont traitées via email sécurisé. Nous recommandons d'utiliser une connexion sécurisée (HTTPS) lorsque vous visitez notre site.

7. Conservation des Données

Vos données personnelles sont conservées :

  • Demandes de procuration : Jusqu'à la fin de la période électorale, puis supprimées
  • Données de navigation : Jusqu'à 13 mois maximum (selon les logs de serveur)
  • Communications email : Tant que nécessaire pour le traitement de votre demande

8. Vos Droits

Conformément au Règlement Général sur la Protection des Données (RGPD), vous avez le droit de :

  • Accès : Demander accès aux données que nous détenons sur vous
  • Rectification : Corriger ou mettre à jour vos données
  • Suppression : Demander la suppression de vos données (dans certains cas)
  • Portabilité : Recevoir vos données dans un format structuré et lisible
  • Opposition : Vous opposer au traitement de vos données
  • Retrait du consentement : Retirer votre consentement à tout moment

Pour exercer l'un de ces droits, contactez-nous à : saussinesnotrevillage@gmail.com

9. Cookies

Notre site web peut utiliser des cookies (petits fichiers stockés sur votre appareil) pour :

  • Améliorer l'expérience utilisateur
  • Mémoriser vos préférences
  • Analyser le trafic du site

Vous pouvez contrôler les cookies via les paramètres de votre navigateur. Le refus de cookies ne devrait pas empêcher l'accès à notre site.

10. Liens Externes

Notre site contient des liens vers des sites externes (notamment Facebook). Nous ne sommes pas responsables des politiques de confidentialité de ces sites. Nous vous recommandons de consulter leur politique de confidentialité avant de partager vos données personnelles.

11. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page avec une date de mise à jour. Nous vous recommandons de consulter régulièrement cette politique.

Dernière mise à jour : 5 mars 2026

12. Contact

Si vous avez des questions concernant cette politique de confidentialité ou nos pratiques en matière de données personnelles, veuillez nous contacter :

  • Email : saussinesnotrevillage@gmail.com
  • Sujet : Indiquez "Confidentialité" dans l'objet de votre email

13. Plainte auprès d'une Autorité de Contrôle

Si vous estimez que vos droits en matière de protection des données ont été violés, vous avez le droit de déposer une plainte auprès de l'autorité de contrôle compétente (CNIL en France). Plus d'informations sur www.cnil.fr

Demande de Procuration

Vous n'êtes pas disponible le jour de l'élection ? L'un de nos volontaires peut porter votre voix.

Pour cela, envoyez-nous un mail à saussinesnotrevillage@gmail.com avec les indications suivantes :

Vos informations

  • Vos prénoms tels qu'indiqués sur votre pièce d'identité
  • Votre nom de naissance
  • Votre numéro de téléphone

Votre demande

  • Votre numéro d'électeur (Le numéro national d'électeur contient jusqu'à 9 chiffres et est présent sur la carte électorale. Il peut également être retrouvé sur le service Interroger sa situation électorale)
  • Votre date de naissance
  • Votre sexe
  • Votre adresse postale électorale
